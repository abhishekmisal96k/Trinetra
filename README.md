# Trinetra 🔍

**Trinetra** is an Android application designed to detect **deepfake content** in videos, audio, and images.  
The goal of this project is to help users identify manipulated media and reduce the spread of misinformation.

## 📱 Overview

Deepfake technology has made it easier to create highly realistic fake media.  
Trinetra aims to provide a simple mobile tool that can analyze media files and determine whether they are authentic or potentially manipulated.

The application is built for Android using **Java**.

## 🚀 Features

- 🖼️ Detect deepfake **images**
- 🎥 Detect deepfake **videos**
- 🎧 Detect deepfake **audio**
- 📂 Upload or select media files for analysis
- 📊 Display detection results in a simple interface
- 📱 Mobile-friendly Android UI

## 🛠️ Tech Stack

- **Language:** Java  
- **Platform:** Android  
- **IDE:** Android Studio

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/trinetra.git
```

2. Open the project in **Android Studio**

3. Build and run the application on an Android device or emulator.

## 📖 Usage

1. Launch the Trinetra application.
2. Select or upload a media file (image, video, or audio).
3. The system analyzes the content.
4. View the detection result indicating whether the media may be a deepfake.

## 🎯 Purpose

The purpose of Trinetra is to provide a tool that helps users verify digital media authenticity and combat misinformation caused by deepfake technology.
